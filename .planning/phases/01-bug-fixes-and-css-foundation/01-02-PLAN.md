**Decision: use index.html#section-id for the 5 section links.** This ensures users navigating from ANY subpage always land on the homepage at the correct section. This matches the convention stated in CONCERNS.md: "Update subpage navbar links to use `index.html#section-id` format".

**Exact replacements in each subpage's navbar-links and mobile-menu:**

| `projekty.html` (nav link) | `index.html#projekty` |
| `index.html#kontakt` | stays as `index.html#kontakt` (already correct) |

Apply to BOTH navbar-links AND mobile-menu in each file.

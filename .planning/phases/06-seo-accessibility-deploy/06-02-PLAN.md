```markdown
# Deploy Guide — Samorząd Studentów UEW

## 1. Vercel (Primary — Live Site)

Vercel is connected to GitHub `main` branch and deploys automatically on push.

### Environment Variables

Set in: Vercel Dashboard -> Project -> Settings -> Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `RESEND_API_KEY` | Yes | Resend.com API key for email forms |
| `RESEND_FROM_EMAIL` | Yes | Verified sender email (e.g. `noreply@samorzad.ue.wroc.pl`) |
| `NEXTAUTH_SECRET` | Yes | Random 32+ char secret — generate: `openssl rand -base64 32` |
| `NEXTAUTH_URL` | Yes | Full URL of site (e.g. `https://samorzad.ue.wroc.pl`) |
| `GOOGLE_CLIENT_ID` | Yes | Google OAuth client ID (console.cloud.google.com) |
| `GOOGLE_CLIENT_SECRET` | Yes | Google OAuth client secret |
| `ANTHROPIC_API_KEY` | Yes | Anthropic API key for AI chatbot |

### Manual Deploy

```bash
# Install Vercel CLI if needed
npm i -g vercel

# Deploy to production
vercel --prod
```

## 2. Static Export (SFTP / Uczelniany Hosting)

Static export disables: API routes, AI chatbot, Google OAuth, email forms.
Use for read-only hosting where API keys cannot be stored.

### Build Static Export

```bash
# Build — produces /out directory
NEXT_STATIC_EXPORT=true npm run build

# Test locally before upload
npx serve out
# Open http://localhost:3000 — verify pages load
```

### Upload via SFTP

Upload the entire contents of the `/out` directory to the web root of the uczelniany hosting.

```
/out/
  index.html          -> /public_html/index.html
  dla-studenta/       -> /public_html/dla-studenta/
  ...
```

SFTP clients: FileZilla, Cyberduck, WinSCP.

### Static Export Limitations

When built with `NEXT_STATIC_EXPORT=true`:
- `/api/*` routes are NOT included — contact forms show an error message
- AI chatbot is NOT functional
- Strefa Dzialacza (Google OAuth) does NOT authenticate — shows landing page only
- All public informational pages work correctly

## 3. Custom Domain

Point DNS to Vercel:
- CNAME: `@` -> `cname.vercel-dns.com`
- Or Vercel nameservers

Configure in: Vercel Dashboard -> Project -> Settings -> Domains

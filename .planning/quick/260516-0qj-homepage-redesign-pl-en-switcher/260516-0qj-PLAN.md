**lib/i18n.tsx** — `'use client'`. Create:
1. `type Lang = 'pl' | 'en'`
2. `type Translations = typeof import('./translations/pl').pl` (derive type from pl object to avoid duplication)
3. `LangContext = createContext<{ lang: Lang; t: Translations; setLang: (l: Lang) => void }>`
4. `LangProvider` component: reads `localStorage.getItem('lang')` as initial lang (default `'pl'`), stores state, writes to localStorage on change, provides context. Wrap children in `<LangContext.Provider>`.
5. `useT()` hook: returns `{ t, lang, setLang }` from context. Throw if used outside provider.

Important: `localStorage` is not available during SSR. Guard with `typeof window !== 'undefined'` or use `useEffect` for initial read. Simplest: initialize state with `'pl'`, then in `useEffect` read localStorage and set if present.
